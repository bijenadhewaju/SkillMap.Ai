# SkillMap AI

SkillMap AI is an AI-powered web application that helps students, graduates, job seekers, and professionals identify skill gaps for their desired career paths. By analyzing users' existing skills and comparing them with career requirements, the platform generates personalized learning roadmaps and recommends relevant courses and projects.

## Features

- User Authentication
- User Profile Management
- Skill Management
- Career Recommendation
- AI Skill Gap Analysis
- Personalized Learning Roadmap
- Course Recommendations
- Project Recommendations
- Progress Tracking Dashboard
- Admin Panel
